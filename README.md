# Crowd Analysis Backend

YOLO-CROWD based real-time crowd detection and analysis API server.

## Features
- Video upload and stream processing
- Real-time crowd density monitoring
- Alert generation for crowd surges
- Per-user job isolation
- MongoDB Atlas integration
